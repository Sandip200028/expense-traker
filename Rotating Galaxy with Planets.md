# Rotating Galaxy with Planets

A visually stunning rotating galaxy simulation built using HTML, CSS, and JavaScript.

## Features
- Rotating Sun at the center
- Planets orbiting at different speeds
- Hover over planets to see their names
- Colorful, animated galaxy effect

## Technologies Used
- HTML Canvas
- CSS
- JavaScript

## How to Run
1. Download or clone this repository
2. Open `index.html` in a browser
3. Enjoy the galaxy simulation!
